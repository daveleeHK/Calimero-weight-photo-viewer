# Calimero Weight-Photo Coverage Viewer

This repository hosts a static research viewer for Calimero, a male African elephant at Safaripark Beekse Bergen.

The viewer links dated institutional live-weight records with dated public photo context. Images are remotely embedded from their original public source pages and are not downloaded or redistributed.

This project is a single-individual research dossier and should not be generalized directly to wild elephant populations. Photo context is descriptive only and does not infer health status.

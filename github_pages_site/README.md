# Calimero Weight-Photo Coverage Viewer

This folder is ready for GitHub Pages publication.

## Publish as a personal GitHub homepage

1. Create or open a public GitHub repository named `<your-github-username>.github.io`.
2. Upload `index.html` and `.nojekyll` from this folder to the repository root.
3. Open `https://<your-github-username>.github.io/` after GitHub Pages finishes deploying.

## Publish as a project page instead

1. Create a public GitHub repository, for example `calimero-weight-photo-viewer`.
2. Upload `index.html` and `.nojekyll` to the repository root.
3. In Settings > Pages, deploy from the main branch root.
4. Open `https://<your-github-username>.github.io/calimero-weight-photo-viewer/`.

## Notes

- Images are remote embedded from public source URLs and are not redistributed in this repository.
- Viewers need internet access.
- If a source site blocks hotlinking or changes image URLs, use each card source link to view the original page.
- The page is visual context only and does not infer health status.
